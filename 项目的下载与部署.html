Bhuman预备篇——项目的下载与部署

本文介绍了如何下载和部署Bhuman框架，特别是针对2019和2017两个版本。对于2019版本，主要涉及依赖项的安装；而对于2017版本，详细阐述了从下载框架到编译项目的过程，包括alcommon的安装和项目编译。

2019版本

在项目编译中：sudo apt install clang make qtbase5-dev libqt5opengl5-dev libqt5svg5-dev libglew-dev libasound-dev ocl-icd-opencl-dev net-tools graphviz xterm

不需要安装alcommon

2017版本

a.下载Bhuman2017框架

        国内大多高校采用的框架是德国的bhuman，此系列以ubuntu系统的bhuman2017为切入点，介绍Robocup比赛所需的准备。百度搜索github进入官网查找项目bhuman，将其下载到电脑上。

        参考链接：https://github.com/bhuman/BHumanCodeRelease

b.项目的创建及编译

        1.在官网下载alcommom包，无需解压，将其放置在Desktop目录下

            下载地址：https://community.ald.softbankrobotics.com

        2.相关软件的下载

             ctrl+alt+t或鼠标右键打开终端，输入：

             sudo apt install clang qtbase5-dev libqt5svg5-dev libglew-dev libxml2-dev graphviz xterm 

             紧接着输入自己的管理员密码，在此期间保持计算机联网

        3.安装官网上下载的压缩包

            将下载好的文件放置在Desktop目录下，进入Install目录，打开终端输入:

            ./installAlcommon ~/Desktop/naoqi-sdk-2.1.4.13-linux32.tar.gz

             官方alcommon目录与上面不一样时，进行相应修改

 

             

         4.项目的编译

            进入解压后的文件夹下的make/linux目录，右键打开终端输入   make   

            一段时间后整个项目部署在电脑中，在根目录下 多出了一个文件夹---build

            在此之后，该系列都将包含build目录的整个项目文件夹称为bhuman
